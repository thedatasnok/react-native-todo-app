## Getting started

### Prerequisites

- PNPM
- Node.js 18 (LTS)
- Android Studio SDK or equivalent tooling for iOS

### Running the project

1. Clone the repository
2. Install dependencies

   ```bash
   pnpm install
   ```

3. Run the app

   ```bash
   pnpm android # or ios
   ```
